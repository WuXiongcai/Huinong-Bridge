# 惠农云桥 - 惠州市龙门县乡村旅游与助农平台

## 项目简介

惠农云桥是一个专注于惠州市龙门县乡村农业发展的综合性平台，由惠州学院在校大学生团队开发。平台致力于通过互联网技术和现代化管理手段，推动龙门县乡村农业和旅游业的可持续发展，助力乡村振兴战略的实施。

目前，平台已成功对接15个村的特色农产品，帮助50多户农民打开线上销售渠道，获得省级创新创业大赛的认可。

## 功能特点

### 1. 前台功能

- **首页展示**：平台概况、最新动态、重要公告
- **乡村文化传播**：展示乡村振兴故事，传递农村发展希望
- **助农农产品购买**：特色农产品在线展示与交易
- **龙门景点推荐**：推广龙门县特色旅游景点与农家乐
- **用户中心**：个人信息管理、订单管理等

### 2. 后台管理

- **用户管理**：管理平台用户信息和权限
- **农产品管理**：农产品信息的增删改查
- **订单管理**：处理和跟踪农产品订单
- **政策资讯管理**：发布最新农业政策与补贴信息
- **系统管理**：平台基础设置和运营数据分析

## 技术架构

### 前端技术栈

- Vue.js：前端框架
- Element UI：UI组件库
- Vue Router：路由管理
- Axios：HTTP请求处理

## 项目结构说明

### 前台模块 (src/views/front/)

- **Home.vue**: 首页组件，展示平台概况和重要信息；包含公告、轮播图展示、助农资讯、助农政策、精选农家乐、农家乐排行榜、帮扶项目、特价农产品购买、合作伙伴等。
- **ProductList.vue**: 农产品列表展示
- **About.vue**: 关于平台的介绍（自由编写）
- **Apply.vue**: 助农申请部分
- **TourismSpots.vue**: 旅游景点推荐
- **UserCenter.vue**: 用户个人中心
- **OrderList.vue**: 用户订单列表
- **ShoppingCart.vue**: 购物车功能

### 后台模块 (src/views/manager/)

- **Dashboard.vue**: 管理后台首页
- **UserManage.vue**: 用户信息管理
- **ProductManage.vue**: 农产品信息管理
- **OrderManage.vue**: 订单处理和管理
- **NewsManage.vue**: 新闻和政策信息管理
- **SystemConfig.vue**: 系统配置管理

### 工具类文件 (src/utils/)

- **request.js**: Axios请求封装，统一处理API请求
- **date.js**: 日期格式化工具
- **amap.js**: 高德地图API集成工具

### 配置文件

- **vue.config.js**: Vue项目配置文件
- **.env.development**: 开发环境配置
- **.env.production**: 生产环境配置
- **babel.config.js**: Babel转译配置
- **package.json**: 项目依赖配置

## 项目运行

### 环境要求

- Node.js >= 12.0.0
- npm >= 6.0.0

### 安装步骤

1. 克隆项目到本地
```bash
git clone [项目地址]
```

2. 进入项目目录
```bash
cd vue
```

3. 安装依赖
```bash
npm install
```

4. 启动开发服务器
```bash
npm run serve
```

5. 打包生产环境
```bash
npm run build
```

### 开发环境配置

项目包含两个环境配置文件：
- `.env.development`：开发环境配置
- `.env.production`：生产环境配置

3. **旅游推广**：推动龙门县乡村旅游发展
